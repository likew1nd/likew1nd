## About Me

我喜欢把复杂的业务流程做成清晰、稳定、可部署的系统。  
目前主要关注 Web 后台、软件授权、卡密验证、用户体系、支付配置和轻量化安装部署。

- 构建基于 ThinkPHP / PHP 的后台系统
- 设计软件授权、卡密、版本与用户管理流程
- 优化安装向导、部署文档和发布包
- 打磨更现代、更顺手的控制台界面
